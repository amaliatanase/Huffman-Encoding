\doxysubsection{Detailed Description}
Represents a binary code as a stack of bits. 


\begin{DoxyItemize}
\item This class is used to dynamically build the binary path (0s and 1s) while traversing the \doxylink{class_huffman}{Huffman} tree. 
\end{DoxyItemize}
