This project implements the {\bfseries{\doxylink{class_huffman}{Huffman}}} {\itshape lossless} data compression and decompression algorithm. It consists of two command-\/line utilities that work together to reduce file sizes and accurately restore them to their original state.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{🚀 Features}{🚀 Features}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Encoder ({\ttfamily encoder})}}:\+ Calculates character frequencies, builds an optimal \doxylink{class_huffman}{Huffman} tree, and writes a compressed binary file ({\ttfamily .huff}) using bit-\/level manipulation (bit-\/packing).
\item {\bfseries{Decoder ({\ttfamily decoder})}}:\+ Reads the header and serialized tree structure from the compressed file to reconstruct the original text. It supports output redirection to {\ttfamily stdout} if no output file is specified.
\item {\bfseries{Detailed Statistics ({\ttfamily -\/s})}}:\+ Both applications can calculate and display the percentage of space saved after compression.
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{🛠️ Compilation}{🛠️ Compilation}}\label{index_autotoc_md2}
